Quick start in 5 minutes.

First time setup

  1. Settings: enter shop name, address, phone numbers.
  2. Vegetables: add products (Tomato, Potato, etc.).
  3. Suppliers: add suppliers with phone numbers.
  4. Customers: walk-in customer already exists.
  5. Day Open: open today to start.

Daily operations

Morning

Day Open → Select Today → Open Day

Purchase stock

Purchases → Select Supplier → Select Item → Bags: "10+20+30" → Add → Save

Make sale

Sales → Select Customer → Select Item → Bags: "10+20" → Add → Save & Print

End of day

Day Close → Review Balance → Adjust if needed → Close Day

Bag quantities made simple.

Use plus-separated bag weights like 10+60+50 to split bill lines.

Beans | 1 | 10.00 | 50.00 | 500.00 Beans | 1 | 60.00 | 50.00 | 3,000.00 Beans | 1 | 50.00 | 50.00 | 2,500.00

Troubleshooting

  • Day is not open: open today from Day Open.
  • Over-selling warning: record purchases or override.
  • Bill not printing: verify printer name in Settings.
  • Database location: %AppData%\VegWholesalePOS\vegshop.db
